getFocusMode

@Nullable open fun getFocusMode(): String

Gets the current focus mode setting.

Return
current focus mode. This value is null if the camera is not yet created. Applications should call to start the focus if focus mode is FOCUS_MODE_AUTO or FOCUS_MODE_MACRO.
