liveExecutions

val liveExecutions: LiveData<MailPackage<List<ServerWithExecutions>>>

LiveData containing executions that belong to a server that is used as a filter, or all execution if no server is set as a filter. It is all wrapper in MailPackage, so it can even represent loading while the executions are being downloaded.
