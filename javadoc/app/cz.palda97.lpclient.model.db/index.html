MarkForDeletion

 data class MarkForDeletion

Entity used to store keys of items determined for deletion. Marks are intended to be stored in database while the UNDO button is visible. If the application is killed before the delete request is sent, the mark alongside with the item for deletion stays in database and the delete request is sent on the next app start.
