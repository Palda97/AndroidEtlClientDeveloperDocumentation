makeItAllWork

fun <itemType> makeItAllWork(recyclerView: RecyclerView, adapterWithList: AdapterWithList<itemType>, deleteFunction: (itemType) -> Unit, context: Context, dividers: Boolean = true, swipeDirection: Int = LEFT + RIGHT): Unit

Adds divider between list items and adds swipe to delete functionality with red color and bin icon.

Parameters

recyclerView - Recycler view to be tuned.

adapterWithList - Adapter that will be assigned with the recycler view.

deleteFunction - Delete function that will be used when swiping to the side.

context - Context.

dividers - If there should be dividers between individual items.

swipeDirection - Direction items will be allowed to be swiped to. Choose from LEFT, RIGHT and their combination (LEFT + RIGHT).
